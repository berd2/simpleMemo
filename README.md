# 메모장

## 개요

* 간단 메모장. 마크다운 지원 및 테마 기능 포함.
* entry point : memo.py

## 사용법

**화면 구성**

메모장은 크게 **왼쪽의 메모 목록 패널**과 **오른쪽의 편집 패널**로 나뉩니다.
![메인화면](memo.png "화면")

*   **왼쪽 패널 (메모 목록)**:
    *   **카테고리 선택**: 상단 콤보박스에서 카테고리를 선택하거나 직접 입력하여 필터링할 수 있습니다.
    *   **검색창**: 메모의 제목이나 내용에 포함된 키워드를 입력하여 메모를 검색할 수 있습니다.
    *   **삭제 취소(⟲) 버튼**: 실수로 삭제한 메모를 복원할 수 있습니다. (최근 10개)
    *   **메모 리스트**: 중요도(📌) 및 순서에 따라 메모 목록이 표시됩니다.
        *   **핀(📌/◻︎)**: 중요한 메모를 상단에 고정합니다.

*   **오른쪽 패널 (편집)**:
    *   **새 메모(🞧) 버튼**: 새로운 빈 메모를 생성합니다.
    *   **저장 (💾) 버튼**: 현재 편집 중인 메모를 명시적으로 저장합니다.
    *   **마크다운 버튼(B, U, H)**: 굵게, 밑줄, 제목(#) 서식을 빠르게 적용합니다.
    *   **메뉴(☰) 버튼**: 폰트, 테마, 줄간격, 새 메모 템플릿 등을 설정합니다.
    *   **편집 영역**: 마크다운 하이라이팅 및 체크박스 토글을 지원합니다. 첫 줄은 자동으로 제목 스타일이 적용됩니다.

**핵심 기능**

*   **자동 저장**: 타이핑 중단 2초 후 또는 다른 메모 선택 시 자동으로 저장됩니다.
*   **마크다운 지원**: `# 제목`, `**굵게**`, `__밑줄__`, `~~취소선~~`, `` `코드` `` 등을 지원합니다.
*   **체크리스트**: `- [ ]` 또는 `- [x]` 클릭 시 상태가 토글됩니다.
*   **다양한 테마**: Light/Dark 테마를 지원하며 설정이 유지됩니다.
*   **휴대성**: 실행 파일 위치나 권한에 따라 AppData 또는 임시 폴더에 DB를 안전하게 생성합니다.

## Guideline

### Project Structure
- `memo.py`: 메인 GUI 애플리케이션.
- `memo_db.py`: SQLite 데이터베이스 핸들러.
- `memo.db`: 로컬 데이터베이스 파일.

### 실행 방법
- `pip install PySide6`
- `python memo.py`
