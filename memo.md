# 사용법

**화면 구성**

메모장은 크게 **왼쪽의 메모 목록 패널**과 **오른쪽의 편집 패널**로 나뉩니다.

*   **왼쪽 패널 (메모 목록)**:
    *   **카테고리 선택/입력**: 상단의 콤보박스에서 기존 카테고리를 선택하여 필터링하거나, 새로운 카테고리를 입력 후 Enter를 눌러 새 메모를 해당 카테고리에 생성할 수 있습니다.
    *   **검색창**: 메모의 제목이나 내용에 포함된 키워드를 입력하여 메모를 검색할 수 있습니다.
    *   **삭제 취소(⟲) 버튼**: 실수로 삭제한 메모를 최근 10개까지 복원할 수 있습니다.
    *   **메모 리스트**: 모든 메모의 목록이 표시됩니다.
        *   **핀(📌/◻︎)**: 중요한 메모를 상단에 고정하는 '핀' 버튼입니다. 핀으로 고정한 메모는 항상 목록의 맨 위에 표시되며, 고정한 순서대로 정렬됩니다.
        *   **제목 및 요약**: 메모의 첫 줄이 제목으로, 그 다음 내용이 요약으로 표시됩니다.
        *   **삭제(☒) 버튼**: 해당 메모를 삭제합니다.

*   **오른쪽 패널 (편집)**:
    *   **위(🡡) / 아래(🡣) 이동 버튼**: 선택한 메모의 순서를 위아래로 변경할 수 있습니다.
    *   **새 메모(🞧) 버튼**: 새로운 빈 메모를 생성합니다.
    *   **저장 (💾) 버튼**: 현재 편집 중인 메모의 내용을 저장합니다.
    *   **마크다운 서식 버튼(B, U, H)**: 선택한 텍스트에 굵게(Bold), 밑줄(Underline)을 적용하거나 현재 줄을 제목(Heading)으로 토글합니다.
    *   **타임스탬프**: 메모가 생성된 시간을 표시합니다.
    *   **메뉴(☰) 버튼**: 오른쪽 상단 끝에 위치하며, 클릭시 왼쪽으로 펼쳐집니다.
        *   **Font**: 메모장의 글꼴, 크기 등을 설정하고 저장합니다.
        *   **Theme**: Light/Dark뿐만 아니라 Mint, Aqua, Navy, Charcoal 등 14가지 다양한 테마를 제공하며 선택 시 즉시 적용되고 저장됩니다.
        *   **Line Spacing**: 메모 본문의 줄간격을 100%, 120%, 150%, 200% 중 하나로 선택할 수 있습니다.
        *   **Load DB**: 다른 경로에 있는 데이터베이스 파일을 불러올 수 있습니다.
        *   **New Memo message**: 새 메모의 초기 텍스트 템플릿을 설정합니다 (날짜 포맷 또는 커스텀 문자열).
        *   **Help**: 내장된 도움말 창을 엽니다.
    *   **편집 영역**: 메모의 전체 내용을 보고 편집하는 공간입니다. 마우스 우클릭을 통해 마크다운 컨텍스트 메뉴를 사용할 수 있습니다.

**핵심 기능**

*   **메모 작성 및 저장**:
    1.  `새 메모(🞧)` 버튼을 눌러 새 메모를 시작합니다.
    2.  편집 영역에 내용을 입력합니다. 첫 줄은 제목이 됩니다.
    3.  `저장 (💾)` 버튼을 누르면 내용이 데이터베이스에 저장됩니다.

*   **자동 저장**:
    *   메모를 편집하던 중 왼쪽 목록에서 다른 메모를 선택하거나, 타이핑을 멈추고 2초가 지나면 현재 편집 중인 내용이 **자동으로 저장**되므로 데이터가 유실될 걱정이 없습니다.

*   **마크다운 및 체크리스트 지원**:
    *   **마크다운 하이라이팅**: `# 제목`, `**굵게**`, `__밑줄__`, `*기울임*`, `~~취소선~~`, `` `코드` `` 등의 문법을 작성하면 색상이 하이라이팅 됩니다. 
    *   **동적 헤더 크기**: `#` 개수에 따라 헤더의 글씨 크기가 동적으로 조절됩니다.
    *   **클릭 가능한 체크리스트**: `- [ ] 할 일` 과 같이 작성 후 편집창에서 체크박스 부분을 클릭하면 `- [x]` 형태로 즉시 토글됩니다.

*   **휴대성 및 안전한 DB 경로**:
    *   실행 파일이 쓰기 권한이 없는 폴더(`Program Files` 등)에 위치한 경우, 자동으로 사용자의 `AppData` (또는 임시 폴더) 영역에 DB를 생성하고 기존 DB를 복사하여 권한 오류 없이 안전하게 실행됩니다.

**새 메모 템플릿 설정**

*   ☰ → **New Memo message**에서 새 메모의 첫 줄을 정할 수 있습니다.
    *   **Date** 하위 메뉴: `YY/MM/DD`, `YYYY-MM-DD` 등 다양한 포맷을 선택하면 새 메모 생성 시 날짜 문자열이 자동 입력됩니다.
    *   **String…**: 직접 문자열을 입력하여 템플릿으로 저장할 수 있습니다. `%D`, `%t`, `%c` 등의 변수를 활용해 카테고리나 날짜를 동적으로 삽입할 수도 있습니다.


** 실행 옵션 ** 

* `memo.py` supports the following command-line arguments:
*   `-db <path>`, `--db <path>`: Specifies the SQLite database file path. Default is `memo.db`.
*   `-scale <factor>`, `--scale <factor>`: Overrides `QT_SCALE_FACTOR` (e.g., `1.1`).
*   `-h`, `--help`: Shows the help message and exits.
